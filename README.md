# 🌐 URL 清理规则云端仓库

> 让你的浏览器链接更清爽、更简洁

---

## 📖 这是什么？

这是一个为 **网站URL简化** 脚本提供支持的云端服务。

简单来说，它能帮你：
- ✅ 自动去除网页链接里的多余参数
- ✅ 让分享的链接更干净、更专业
- ✅ 无需手动更新，规则自动同步

---


### 📱 支持哪些网站？

目前支持以下网站的链接清理：

| 网站 | 说明 |
|------|------|
| 🔍 必应搜索 | 去除搜索时的追踪参数 |
| 🔍 百度搜索 | 简化搜索结果链接 |
| 📺 B站视频 | 去除视频分享中的多余参数 |
| 🤖 KIMI AI | 清理对话页面链接 |
| 🔍 360搜索 | 简化搜索链接 |
| 📝 CSDN博客 | 去除文章页面的追踪代码 |
| 🔍 搜狗搜索 | 简化搜索链接 |
| 🔍 Google搜索 | 去除搜索追踪参数 |

## 🚀 如何使用？

### 第一步：安装脚本

1. 安装浏览器扩展 **Tampermonkey**（篡改猴）
   - [Chrome 商店](https://chrome.google.com/webstore/detail/tampermonkey)
   - [Firefox 商店](https://addons.mozilla.org/firefox/addon/tampermonkey)

2. 安装 **网站URL简化** 脚本
   - 从 [Greasy Fork](https://greasyfork.org) 搜索安装
   - 或从 [脚本猫](https://scriptcat.org) 安装

### 第二步：启用云端更新

1. 在网页上点击 Tampermonkey 图标
2. 找到 **网站URL简化** 脚本
3. 点击 **"⚙️ 清理规则设置"**
4. 打开 **"启用云端规则更新"** 开关
5. 点击保存

### 第三步：选择更新方式

我们提供三种更新方式，你可以根据需要选择：

| 方式 | 特点 | 推荐 |
|------|------|------|
| **稳定版** | 最可靠，每月更新 | ⭐ 推荐大多数用户使用 |
| **测试版** | 新功能尝鲜，每周更新 | 适合喜欢尝试新功能的用户 |
| **开发版** | 最新功能，每天更新 | 适合愿意帮助测试的用户 |

## 💡 使用场景

### 场景一：分享视频链接

当你在 B 站看到喜欢的视频，想分享给朋友时：

**以前：**
> 复制出来的链接很长，带着一堆看不懂的参数

**现在：**
> 链接自动变简洁，朋友点开直接看视频，没有多余跳转

### 场景二：收藏搜索页面

当你在搜索引擎找到想要的结果，想保存链接时：

**以前：**
> 链接里带着搜索记录、广告追踪等信息

**现在：**
> 链接只保留核心搜索词，清爽干净

### 场景三：整理资料

当你在做资料收集，需要记录大量网页链接时：

**以前：**
> 链接又臭又长，整理起来很麻烦

**现在：**
> 所有链接自动简化，方便管理和分享

## ❓ 常见问题

### Q: 这个功能安全吗？

**A:** 完全安全。我们只做一件事：去除链接中多余的参数。不会收集你的个人信息，不会修改网页内容，不会影响正常浏览。

### Q: 为什么有些链接清理不了？

**A:** 可能有以下原因：
- 该网站尚未支持（可以向我们反馈）
- 链接格式特殊
- 网络连接问题

如果遇到无法清理的链接，欢迎向我们反馈。

### Q: 更新失败怎么办？

**A:** 请尝试以下方法：
1. 检查网络连接是否正常
2. 切换到其他更新方式（如从测试版切换到稳定版）
3. 重启浏览器后再次尝试

如果问题仍然存在，请通过下方反馈渠道联系我们。

### Q: 如何开启/关闭云端更新？

**A:** 
1. 点击 Tampermonkey 图标
2. 选择 **网站URL简化** 脚本
3. 点击 **"⚙️ 清理规则设置"**
4. 关闭 **"启用云端规则更新"** 开关
